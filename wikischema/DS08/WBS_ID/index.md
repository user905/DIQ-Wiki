| ------ | ----- |
| type | string |
| key | True |
| description | CA or SLPP WBS level identifier. |
| minLength | 1 |
| maxLength | 150 |
| $comment | WBS ID is has a max length of 50 for all new projects. Max length of 150 is enforced here to support specific legacy projects only. |
